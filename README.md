# Smart Home Mobile App

## Local Environment

### Software
* **NPM:** 8.19.1
* **Node:** 18.9.0
* **Expo:** 6.3.2

### Developing
#### Setup
```
npm install
```
#### Start the App
```
 expo start --tunnel
```

### Start App with API url
```
API_URL="${IP-ADDRESS}" npx expo start
```
### Run Jest Tests
```
npm run test
```

### Run Linter
```
eslint .
```

### Fix Linter errors
```
eslint . --fix
```
